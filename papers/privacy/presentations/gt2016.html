Comment on extra material

The extra material contains some Python code. If you want to find out what this code means, check this simple tutorial on the quantitative economics website. If you want to run the code on your machine (this allows you to try other games than the ones I use as examples), you need to have Python (with the openopt package) installed. You can check the quantitative economics website for a guide how to setup and use Python. Note that Python is by default installed on Linux systems (also on Mac?). I use Python 2.7 and the programs might not run in Python 3. To install the openopt package, try (after installing Python – that is after installing Anaconda if you follow the guide) the following in your terminal (or "cmd" if you are running Windows): 

pip install openopt
pip install SpaceFuncs
pip install FuncDesigner
pip install DerApproximator

If you are running linux, you might have to add "sudo" in front.

  • some disclaimers

    I cannot guarantee that the code in the extra material works flawlessly or produces correct results (though I test it a little bit). Even if it works, the code is very far from being efficient. That is, a proper programmer could write programs that do the job much better and faster. If you are interested in somewhat more professional programs to analyze games, you could check out gambit.

  • FAQs regarding the "extra" material:
    • Is the extra material relevant for the exam? Only if it was used directly in the lecture. The code is absolutely irrelevant for the exam.
    • Why should I care about the extra material? You do not have to! But…
      • if you want to get some extra practice it might come in handy. You can make up your own exercises and use the programs to check for the correct solutions.
      • some of the material gives you a slightly different angle on things we did in the lecture. This might help you to better understand the topic.
      • playing around with numerical examples helps some people to get a better idea of the problem.
      • being able to go through some basic code is a skill valued in the labor market.
    • Is it safe to install the software mentioned above? All the software mentioned here is free and open source software. Python and the openopt suite are widely used in industry and science. Having said that: All code is potentially malicious and I cannot take any responsibility for what happens when you install or use the software.
